# 工作流 1：作业批改

## 触发方式
老师粘贴学生的命令或脚本，并说明作业要求

## 流程

```
[用户输入：脚本/命令 + 作业题目背景]
       │
       ▼
[判断类型] ── 命令操作题 ──→ [分析命令：语法 → 逻辑 → 效率]
       │
       ├── Shell 脚本题 ──→ [分析脚本：语法检查 → 逻辑审查 → 风格审查]
       │
       ▼
[输出：总分 + 三维度数 + 逐条问题 + 修改建议]
```

## Coze 配置要点

1. **触发**：自然语言触发，当用户提到"批改""帮我看看""这个作业"等关键词
2. **代码执行器**：如果代码执行器可用，尝试运行脚本验证输出
3. **知识库检索**：自动检索「评分标准」和「常见错误」辅助判断
