# 测试用例

## 用例 1：批改 Shell 脚本作业

### 题目背景
老师布置作业：编写一个脚本，自动备份 /var/log 目录到 /backup，文件名带日期，超过 7 天的备份自动删除。

### 学生提交（含错误的脚本）
```bash
#!/bin/bash

SRC=/var/log
DST = /backup

mkdir $DST 2>/dev/null

tar -czf $DST/backup.tar.gz $SRC

find $DST -name "*.tar.gz" -mtime +7 -exec rm {} \;

echo "done"
```

### 预期批改结果
应指出以下问题：
1. `DST = /backup` 变量赋值有空格，语法错误
2. 备份文件名没有日期，每次会覆盖
3. 没有用引号包裹变量路径
4. tar 不会因为路径错误而失败但会产生奇怪结果
5. 没有检查 tar 是否成功
6. find -exec rm 应该用 -delete 更安全
