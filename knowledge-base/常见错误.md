# Linux 学生常见错误记录

## 权限相关
1. **chmod 777 滥用** — 遇到权限问题就 777，不懂递归和精准赋权
2. **权限符号混淆** — 搞不清 `rwx` 对应文件和目录的不同含义
3. **忘记 sudo** — 普通用户执行需要 root 权限的操作
4. **脚本忘记 chmod +x** — 写了脚本但没加执行权限

## 路径与文件操作
5. **相对路径误用** — 脚本中用了相对路径，换个目录执行就报错
6. **空格未转义** — 文件名含空格时忘记引号或转义
7. **rm 误删** — `rm * .txt`（想删所有 txt，结果星号把别的也删了）
8. **cp 覆盖不提醒** — 不知道 cp 默认覆盖，丢失原文件

## 管道与重定向
9. **> vs >> 混淆** — 想追加却用了覆盖
10. **2>&1 位置错误** — 写成 `2>&1 > file` 而不是 `> file 2>&1`
11. **cat 滥用** — `cat file | grep xxx` 而不是 `grep xxx file`
12. **管道中变量不传递** — 在管道右侧的 while 里修改变量，左侧读不到

## Shell 脚本
13. **变量赋值空格** — `name = "abc"` 而非 `name="abc"`
14. **[ ] 内忘记空格** — `[$a -eq 5]` 而非 `[ $a -eq 5 ]`
15. **[ vs [[ 混用** — 在 sh 中用 [[（bash 特性），导致不兼容
16. **for 循环中的单词分割** — `for i in $(cat file)` 按单词而非行迭代
17. **变量未加引号** — `if [ -f $file ]` 当 file 为空时报错
18. **exit code 不检查** — 关键命令执行后不检查 $?

## 进程管理
19. **kill -9 滥用** — 不管什么进程都用 -9，不用默认信号优雅停止
20. **后台任务以为关了终端还在跑** — 没用 nohup 或 screen/tmux

## 包管理
21. **混用 apt 和 dpkg** — 不了解两者的关系和适用场景
22. **安装不加 -y 搞不清** — 脚本自动化时忘记 -y 导致卡住

## 网络
23. **ifconfig 找不到** — 不知道新版 Linux 已换成 ip 命令
24. **SSH 端口搞错** — 改了 SSH 端口但防火墙没放行
