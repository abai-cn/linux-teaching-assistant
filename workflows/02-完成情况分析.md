# 工作流 2：批量完成情况分析

## 触发方式
老师上传多份作业文件或粘贴成绩表格

## 流程

```
[用户上传多份作业 / 粘贴成绩数据]
       │
       ▼
[解析数据] → 提取每份作业的分数、提交人、主要问题
       │
       ▼
[统计分析] → 提交率、分数分布、高频错误 TOP 3
       │
       ▼
[生成报告] → 统计表格 + 可视化描述 + 教学建议
```

## 输出模板

```
## 作业完成情况报告

### 总体数据
- 应提交：XX 人
- 实际提交：XX 人
- 提交率：XX%
- 平均分：XX

### 分数分布
| 等级 | 人数 | 占比 |
| 优秀（90+）| X | X% |
| 良好（75-89）| X | X% |
| 及格（60-74）| X | X% |
| 不及格（<60）| X | X% |

### 高频错误 TOP 3
1. [错误描述] — X人涉及
2. ...
3. ...

### 建议
[根据分析结果给出教学建议]
```
