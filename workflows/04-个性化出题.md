# 工作流 4：个性化出题

## 触发方式
老师指定：知识点 + 难度 + 题量（+ 可选：具体场景）

## 流程

```
[用户输入：语言/环境 + 知识点 + 难度 + 题量]
       │
       ▼
[检索知识库] → 获取该知识点的教学大纲和题库模板
       │
       ▼
[生成题目] → 按难度级别生成场景化实验题
       │
       ▼
[自检] → 验证题目本身逻辑自洽、参考答案正确
       │
       ▼
[输出] → 题目 + 评分要点 + 参考答案
```

## 题目类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 命令行操作 | 完成一系列文件/权限/进程操作 | "在 /tmp 下创建目录结构..." |
| Shell 脚本 | 编写脚本解决实际问题 | "写一个日志清理脚本..." |
| 故障排查 | 给定错误现象修复 | "Apache 启动失败，请排查..." |
| 综合实验 | 多知识点组合 | "部署一个简单的 Web 服务..." |

## 输入示例

> 出 2 道 Shell 脚本题，知识点是 cron 定时任务 + 日志处理，难度基础，做出来大概 15 分钟

```

**难度：** 基础 | **预计用时：** 8 分钟 | **知识点：** cron, 日志分析

**题目描述：**
[场景化的问题描述]

**要求：**
1. [具体要求 1]
2. [具体要求 2]

**评分要点：**
- [ ] 评分项 1（X分）
- [ ] 评分项 2（X分）

<details>

[参考答案命令或脚本]

</details>
```
